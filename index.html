🐙 GitHub — Configurar Token

🔄 Verificando token...
⚠️ Gere um token SEM EXPIRAÇÃO:
1. Acesse github.com → foto do perfil → Settings
2. Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token (classic)
4. Em Expiration selecione "No expiration" ← IMPORTANTE
5. Marque o escopo "repo" → clique Generate token
6. Copie e cole o token aqui
